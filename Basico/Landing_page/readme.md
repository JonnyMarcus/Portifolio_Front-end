# Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar os fundamentos de HTML e CSS através da criação de uma Landing Page moderna, organizada e responsiva.

A aplicação possui foco em:
- estruturação semântica
- organização visual
- responsividade
- estilização moderna
- boas práticas de Front-End

---

# Objetivos

- Aprender HTML semântico
- Praticar CSS moderno
- Trabalhar com Flexbox
- Melhorar organização de layout
- Criar componentes reutilizáveis
- Desenvolver responsividade
- Aprender estruturação profissional de projetos
