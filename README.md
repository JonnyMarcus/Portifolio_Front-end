# Sobre o Projeto

Este repositório foi criado com o objetivo de documentar minha evolução no desenvolvimento Front-End utilizando HTML e CSS.

O portfólio está dividido em três níveis:
- Básico
- Intermediário
- Avançado

Cada projeto foi desenvolvido para praticar conceitos específicos de estruturação, estilização, responsividade e criação de interfaces modernas.

---

# Objetivos

- Aprender HTML e CSS na prática
- Desenvolver interfaces modernas
- Melhorar responsividade
- Criar layouts profissionais
- Praticar organização de projetos
- Evoluir no desenvolvimento Front-End
- Aprender boas práticas de UI/UX
- Construir um portfólio visual profissional
